# atropos-demo

A demo app showing how to create AppleTV UI animation using AtroposJS

## Prerequisites

- NodeJS 16.x
- yarn (`npm install -g yarn`)

## Install & Run

```
$ git clone https://github.com/valnub/atropos-demo.git
$ cd atropos-demo
$ yarn
$ yarn start
```
